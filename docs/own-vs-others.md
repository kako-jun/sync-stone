# 自分のブログ vs 他人のブログ

ロドストにログインした状態で、閲覧しているブログが自分のものか他人のものかを自動判定し、動作が異なる。

## 判定方法

```typescript
// body要素に id="community" があれば他人のブログ
const isOwnBlog = !document.body.hasAttribute('id') ||
                  document.body.getAttribute('id') !== 'community';
```

## 動作の違い

| 項目 | 自分のブログ | 他人のブログ |
|------|-------------|-------------|
| **Phase 0（画像一覧収集）** | 実行（`/my/image/`にアクセス） | スキップ |
| **取得できる画像** | 画像一覧 + 記事内画像 | 記事内画像のみ |
| **ZIPファイル名** | `lodestone_complete_export.zip` | `lodestone_others_complete_export.zip` |
| **確認ダイアログ** | `（自分の記事）` | `（自分以外の記事）` |

## 理由

- `/my/image/`（画像一覧ページ）は自分専用のページであり、他人のアカウントではアクセスできない
- 自分のブログの場合、記事に直接埋め込まれていない画像（過去にアップロードしたが記事では使っていない画像など）も含めて全てバックアップ可能
- 他人のブログの場合、記事本文に含まれる画像のみが対象

## 処理フロー

### 自分のブログの場合

```
1. Phase 0: /my/image/ から全画像URLを収集・ダウンロード
2. Phase 1: 各記事ページから詳細情報を抽出
3. Phase 2: 記事内で新たに見つかった画像をダウンロード
4. Phase 3: 統合ZIP生成
```

### 他人のブログの場合

```
1. Phase 0: スキップ
2. Phase 1: 各記事ページから詳細情報を抽出
3. Phase 2: 記事内の画像をダウンロード
4. Phase 3: 統合ZIP生成
```
